Cache Bank

Responsive Banking

A full fledged website using HTML, TailwindCSS, and vanilla JavaScript. With this website I created a vibrant and responsive banking platform.

The website is styled using TailwindCSS classes, with some of my responsive mobile menu functionality being handled by JavaScript.

As seen above, I used hover and focus effects throughout, emphaising both strong design principles and respect for accessibility.

In addition to this, a core theme was chosen for the websites design and this is seen throughout the various sections.

Counters also reload numbers responsively on page load to add a nice aesthetic touch to potential customers

Finally as seen above the website is responsive from mobile resolutions all the way up to desktop.

Additionally, we have a mobile menu in smaller screens to aid in navigation.
